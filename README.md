# CompanieTransport
Student: Ungurean Dorin Samuel
FIESC CALCULATOARE Grupa 3123A

Cerinta temei alese: Să se realizeze o aplicație pentru o companie de transporturi. Se va considera evidența conducătorilor auto, a mașinilor disponibile și a intervalelor de lucru pentru fiecare.
 Opțiuni suplimentare: 
•	Numărul de kilometri parcurși de o mașina / conducător auto
•	Care a fost traseul unui conducător auto
Pentru realizarea acestui proiect se vor crea un numar de clase precum  Firma(care va memora masinile, angajatii, coletele, locuri inchiriate, depozite), Sofer(va memora tipul de transport, salariul, masina, orarul), Masina(va memora capacitatea, distanta de parcurgere, transporturi effectuate), Client(va memora cu ajutorul unui ID  coletele depuse sau locurile inchiriate, destinatie, etc. )
Programul va memora intrarile in fisiere externe citind si scriind atat de la tastatura cat si In fisiere de memorare. Programul va mosteni datele de la o clasa la alta efcientzand solutionarea cerintelor, va arunca exceptii pentru situatiile neprevazute sau pentru erorile de toate felurile.
